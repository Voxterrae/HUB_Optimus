## How to run a simulation (recommended format)
**Preparation (2-5 min)**
- Define roles (Party A, Party B, mediator/observer).
- Define "minimum success" (what conditions make the round worthwhile).
- Define limits (red lines and negotiable zone).

**Execution (3 rounds)**
- Round 1: initial proposal -> response
- Round 2: adjustments (concessions, verification, sequence)
- Round 3: closure (draft agreement + open points)

**Close (5 min)**
- Assess: clarity, verifiability, viability.
- Record: concessions, risks, follow-up conditions.
- Decide: what to test differently in the next iteration?
