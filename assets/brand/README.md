# HUB_Optimus Brand Assets

This directory contains visual onboarding assets for HUB_Optimus.

These assets support recognition, orientation, and README presentation. They do not define governance, roadmap, runtime behavior, project authority, or capability claims.

## Usage rules

- Use repository-relative paths.
- Use `hero/hub-optimus-hero.jpg` in the README unless a reviewer explicitly requests another variant.
- Do not hotlink external images.
- Do not add font files.
- Do not use these assets to imply certification, authority, prediction, or endorsement.
- Keep visual assets aligned with the project principle: reality, evidence, inference, narrative, and operational signal remain separate.

## Canonical meaning

Canonical project meaning remains in the versioned documentation and governance files. Visual assets are presentation support only.
